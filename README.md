# 🦸‍♂️ Superhero Search App

An app developed in **Flutter Web** that allows you to search for superheroes using the [SuperheroAPI](https://superheroapi.com/) API. Designed as a practical way to consume APIs, deploy to GitHub, and handle real-world errors in web environments.
---
NOTE: The app is not deployed in Vercel due to API errors.
---

## 🚀 Features

- 🔍 Search for superheroes by name
- 📊 Display stats like strength, speed, intelligence, etc.

- 🖼️ Responsive and user-friendly interface
- ⚠️ Error handling when consuming the API
